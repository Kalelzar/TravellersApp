virtual std::unique_ptr<Nullable<A> > IList< A >::shift ( )
pure virtual

Removes the first element of the list, if any, and returns it.

This is done by returning a pointer to a Nullable instance. It is an instance of Null if the list is empty or an instance of NotNull wrapped around the value.

All following elements are moved back by one index to fill the hole left by the removal
