std::shared_ptr< ArrayList< Token > > Travel::Scanner::scanNext ( )

Scans the next line in the input stream.

Due to technical limitations the lines are assumed to be of 512 characters or less.

The last token of the returned list is always of type {EOF} a shared pointer to a list containing the resulting tokens
