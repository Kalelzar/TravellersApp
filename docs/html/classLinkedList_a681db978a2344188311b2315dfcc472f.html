std::unique_ptr<Nullable<A> > LinkedList< A >::removeAt ( unsigned  ind)
inlineoverridevirtual

Removes the value associated with the given index, if any, and returns it.

This is done by returning a pointer to a Nullable instance. It is an instance of Null if the index doesn't exist or an instance of NotNull wrapped around the value.

All following elements are moved back by one index to fill the hole left by the removal
