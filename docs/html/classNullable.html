Detailed Description

template<class A>
class Nullable< A >

An optional value container inspired by Haskell's Maybe and Scala's Option.

It stores either a value of type A (wrapped into a NotNull instance) or an empty (represented by Null) value. It is intended to provide more safety when returning empty values than returning nullptr would. It not mean you shouldn't check if a value is null. Rather the entire purpose of the class is for when you do forget to check the value, you don't end up spending 2 hours finding out which nullptr you tried to dereference.
