template<typename B , typename A >
void BinaryNode< B, A >::populate ( BinaryNode< B, A > *  node)
inline

Adds the node to the binary node structure.

If the provided node's tag matches this node's tag add it's values to this node's values. If the provided node's tag is lesser than this node's tag attempt to add the node to this node's left child if such exists, else set it as the left child. If the provided node's tag is greater than this node's tag attempt to add the node to this node's right child if such exists, else set it as the right child.
